# Calculadora de Dieta

Projeto pessoal desenvolvido em HTML, CSS e JavaScript puro para organizar refeições da semana e facilitar o planejamento das compras.

A ideia nasceu de uma necessidade prática: montar as refeições da semana de forma simples, registrar a gramatura dos ingredientes de cada refeição e somar os totais para ajudar no momento de comprar os itens necessários. Como este é um projeto em evolução, ele também serve como exercício de aprendizado e prática, com a proposta de ser construído "na unha", do zero, para fortalecer minha experiência com HTML, CSS e JS.

## Objetivo

O foco da aplicação é ajudar no controle das refeições semanais, permitindo:

- organizar as refeições por dia da semana;
- adicionar ingredientes e quantidades em cada refeição;
- acompanhar os itens cadastrados em uma interface visual estilo kanban;
- evoluir o projeto de forma prática, enquanto estudo e aprimoro minhas habilidades no front-end.

## Como funciona

1. O plano semanal é criado com os dias da semana.
2. O usuário seleciona um dia e cria uma refeição.
3. A refeição recebe ingredientes e quantidades por meio do formulário.
4. A interface é atualizada para exibir os dados cadastrados.

## Como executar

Este projeto não usa build nem dependências externas. Basta abrir o arquivo [index.html](index.html) em um navegador moderno.

Se preferir, você também pode usar uma extensão como Live Server no VS Code para recarregar a página automaticamente durante o desenvolvimento.

## Minha proposta com esse projeto

Além da utilidade prática no dia a dia, este repositório faz parte da minha evolução como iniciante. A intenção é construir tudo do zero, entendendo bem cada parte do processo, para ganhar experiência real com estruturação de páginas, lógica de programação e organização de código front-end.
